\chapter{Introducción}
En este trabajo elaboramos una propuesta de estado del arte del área de \gls{sig:BI} y específicamente Data Discovery. El objetivo es aplicar los conocimientos adquiridos al entorno nacional, en este caso aplicados a una institución del estado \gls{sig:ANDE}, para el apoyo en la toma de decisiones.

Aplicamos las técnicas de Data Discovery a los datos de dos instituciones del estado, específicamente la \gls{sig:ANDE} y \gls{sig:DGEEC}, donde mostramos que con datos de calidad se pueden detectar oportunidades que nos faciliten la toma de decisiones en la institución. Utilizamos conjuntos de datos de las instituciones mencionadas más arriba para este fin.\\

\noindent Este trabajo está estructurado de la siguiente forma:\\

\begin{itemize}[noitemsep, nolistsep]	
	
\item Planteamiento del problema.

\item Objetivos generales y específicos.

\item Justificación.

\item Marco teórico. Se compone de conceptos, componentes, tecnologías de BI, el framework de Gartner, selección de la herramienta para Data Discovery y se explica el cuadrante mágico de Gartner, el cual apoyó a la selección de la herramienta mencionada.

\item Resultado: Diferentes paneles indicadores (dashboards) realizados con la herramienta seleccionada para aplicar data discovery sobre los conjuntos de datos existentes. Además se presentan ejemplos de análisis de datos y la información obtenida, como resultado de la combinación del uso de las herramientas y datos.
\end{itemize}
