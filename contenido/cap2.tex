A continuación se presenta el Cuadrante Mágico de Gartner, para herramientas de BI y Analytics.

Los valores posibles van del 1 al 5, conforme la siguiente evaluación: 

\begin{enumerate}
	\item Pobre o ausente: la mayoría de los requisitos de esta capacidad no fueron alcanzadas.
	\item Justo: Algunos de los requisitos fueron alcanzados.
	\item Bueno: cumple con los requisitos.
	\item Excelente: alcanza o excede algunos requisitos.
	\item Superior: excede significativamente los requisitos.
\end{enumerate}

De una forma ágil el usuario puede conectarse a diversas fuentes de datos y crear paneles interactivos, conectando entre sí los diferentes componentes (tipo de gráfico) que proporciona la herramienta. La herramienta permite utilizar componentes como filtros, siendo o no de la misma fuente de datos siempre y cuando los datos coincidan en los diversos conjuntos.
Pueden ser utilizadas en la organizaciones para comprender rápidamente diferentes aspectos del negocio. También se puede utilizar para realizar proyecciones o tendencias,  la cual Tableau nos ofrece de manera automática.
