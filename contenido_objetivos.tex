\chapter{Objetivos}

\section{Objetivo General}
Aplicar las técnicas de Data Discovery a datos de dos instituciones del estado: \gls{sig:ANDE} y \gls{sig:DGEEC}.

\section{Objetivos Específicos}
\begin{itemize}

	\item Estudiar el estado del arte de \gls{sig:BI} y Data discovery.
	\item Seleccionar la herramienta adecuada para este trabajo.
	\item Analizar y cruzar datos para descubrir situaciones de interés.
	\item Elaborar \gls{sig:Dashboard} que reflejen el análisis de los datos con la herramienta seleccionada.
	\item Elaborar gráficos de tendencias, proyecciones de consumo de energía eléctrica y del crecimiento de los clientes de la institución. 
	\item Proponer algunos \gls{sig:Dashboard} para demostrar que con los datos proveídos es posible detectar situaciones, y estimar o visualizar acontecimientos de interés para una organización.
\end{itemize}
