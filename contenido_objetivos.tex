\chapter{Objetivos}

\section{Objetivo General}
Aplicar las técnicas de Data Discovery a datos de dos instituciones del estado: \gls{sig:ANDE} y \gls{sig:DGEEC}.

\section{Objetivos Específicos}
\begin{itemize}

	\item Estudio del estado del arte de \gls{sig:BI} y Data discovery.
	\item Selección de la herramienta adecuada para este trabajo.
	\item Análisis y cruzamiento de datos para el descubrimiento de situaciones de interés.
	\item Elaboración de \gls{sig:Dashboard} que reflejen el análisis de los datos con la herramienta seleccionada.
	\item Elaboración de gráficos de tendencias, proyecciones de consumo y clientes de la institución. 
	\item Proponer algunos \gls{sig:Dashboard} para demostrar que con los datos proveídos es posible detectar situaciones, y estimar o visualizar acontecimientos de interés para una organización
\end{itemize}
